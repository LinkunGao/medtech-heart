Introduction
===============

The purpose of this project is to convert existing legacy Heart Application to a new website based on modern javascript frameworks. The new website, referred to as  HEART VUE APP is aimed to remove the pitfalls associated with old technology, Thus, it provides enhanced user experience with responsive design, as well as scalability and flexibility to extend it for different subjects.

The following sections provide information regarding the implementation of  HEART VUE APP. This document is aimed to facilitate knowledge transfer and help other team members  to understand the code structure and design choices.
